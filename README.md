# PPT 排版图片生成器

将 PPT 导出的多张图片（16:9），自动排版成适合小红书等平台发布的拼接图片。

## 布局样式

### Layout 1：左侧9缩略图 + 右侧3大图

适合 12 页 PPT，左侧展示前 9 页缩略图，右侧重点展示后 3 页大图。

```
┌──────────────────────────────────────┐
│ ┌────┐   ┌────────────────────────┐ │
│ │ p1 │   │                        │ │
│ ├────┤   │       p10 大图           │ │
│ │ p2 │   │                        │ │
│ ├────┤   ├────────────────────────┤ │
│ │ p3 │   │                        │ │
│ ├────┤   │       p11 大图           │ │
│ │ p4 │   │                        │ │
│ ├────┤   ├────────────────────────┤ │
│ │ p5 │   │                        │ │
│ ├────┤   │       p12 大图           │ │
│ │ p6 │   │                        │ │
│ ├────┤   │                        │ │
│ │ p7 │   │                        │ │
│ ├────┤   │                        │ │
│ │ p8 │   │                        │ │
│ ├────┤   │                        │ │
│ │ p9 │   │                        │ │
│ └────┘   └────────────────────────┘ │
└──────────────────────────────────────┘
```

### Layout 2：三层瀑布流

第一层 1 张大图 + 第二层 2 张横排 + 第三层 2 张横排，适合 5 页 PPT。

```
┌──────────────────────────────────────┐
│                                      │
│           p1 大图（撑满宽度）          │
│                                      │
├──────────────────┬───────────────────┤
│                   │                   │
│      p2           │       p3          │
│                   │                   │
├──────────────────┴───────────────────┤
│                   │                   │
│      p4           │       p5          │
│                   │                   │
└──────────────────────────────────────┘
```

### Layout 3：上下双图 + 标题文字

上方 1 张大图 + 中间标题（居中）+ 下方 1 张大图，适合 2 页 PPT。

```
┌──────────────────────────────────────┐
│                                      │
│           p1 大图（撑满宽度）          │
│                                      │
│         2026小班春季开学主题班会        │  ← 居中标题
│                                      │
├──────────────────────────────────────┤
│                                      │
│           p2 大图（撑满宽度）          │
│                                      │
└──────────────────────────────────────┘
```

## 安装

```bash
# 克隆仓库
git clone https://github.com/kenkl1/dp-xhsppt-output_layout.git

# 进入目录
cd dp-xhsppt-output_layout
```

---

## 使用方法

```bash
# 列出所有布局
python ppt_layout.py --list

# 使用 layout1（默认）
python ppt_layout.py "G:/PPT导出图片/"

# 指定布局
python ppt_layout.py "G:/PPT导出图片/" -l layout2

# 指定布局 + 标题文字（用于 layout3）
python ppt_layout.py "G:/PPT导出图片/" -l layout3 -t "2026小班春季开学主题班会PPT"

# 自定义背景色
python ppt_layout.py "G:/PPT导出图片/" -b "#f3dfc6"

# 自定义左右数量（覆盖配置，用于 layout1）
python ppt_layout.py "G:/PPT导出图片/" --left 6 --right 6

# 指定输出路径
python ppt_layout.py "G:/PPT导出图片/" -o "G:/输出/result.png"
```

### 参数说明

| 参数 | 说明 |
|------|------|
| `input_dir` | 输入图片目录（必需） |
| `-o, --output` | 输出文件路径 |
| `-l, --layout` | 布局类型：`layout1`、`layout2`、`layout3` |
| `-t, --title` | 标题文字（用于 layout3） |
| `-b, --bg` | 背景色，支持颜色名或 HEX，如 `#f3dfc6` |
| `--left` | 左侧图片数量（覆盖配置） |
| `--right` | 右侧图片数量（覆盖配置） |
| `--list` | 列出所有可用布局 |

---

## 输入要求

- 图片格式：PNG、JPG、JPEG
- 图片尺寸：建议 16:9
- 图片命名：按顺序排列

---

## 输出

- 格式：PNG
- 尺寸：1242 × 1660 px
- 位置：与输入图片同目录（可通过 `-o` 指定）

---

## 配置

布局参数通过 `layouts.json` 管理，可自由添加或修改：

```json
{
  "canvas": { "width": 1242, "height": 1660 },
  "defaults": {
    "margin": 40,
    "gap": 20,
    "background": "#f3dfc6"
  },
  "layouts": {
    "layout1": {
      "name": "左侧9缩略图 + 右侧3大图",
      "description": "适合12页PPT",
      "type": "left_right",
      "left": { "count": 9, "width_max": 280 },
      "right": { "count": 3 }
    }
  }
}
```
