# PPT 排版图片生成器

将 PPT 导出的多张图片，自动排版成适合小红书等平台发布的拼接图片。

## 布局样式

### layout1：左侧9缩略图 + 右侧3大图

适合 12 页 PPT，左侧展示前 9 页缩略图，右侧展示后 3 页大图。

```
┌──────────────────────────────────────┐
│ ┌────┐   ┌────────────────────────┐ │
│ │ p1 │   │                        │ │
│ ├────┤   │       p10 大图           │ │
│ │ p2 │   │                        │ │
│ ├────┤   ├────────────────────────┤ │
│ │ p3 │   │                        │ │
│ ├────┤   │       p11 大图           │ │
│ │ p4 │   │                        │ │
│ ├────┤   ├────────────────────────┤ │
│ │ p5 │   │                        │ │
│ ├────┤   │       p12 大图           │ │
│ │ p6 │   │                        │ │
│ ├────┤   │                        │ │
│ │ p7 │   │                        │ │
│ ├────┤   │                        │ │
│ │ p8 │   │                        │ │
│ ├────┤   │                        │ │
│ │ p9 │   │                        │ │
│ └────┘   └────────────────────────┘ │
└──────────────────────────────────────┘
```

---

### layout2：三层瀑布流

第一层 1 张大图 + 第二层 2 张横排 + 第三层 2 张横排，适合 5 页 PPT。

```
┌──────────────────────────────────────┐
│                                      │
│           p1 大图（撑满宽度）          │
│                                      │
├──────────────────┬───────────────────┤
│                   │                   │
│      p2           │       p3          │
│                   │                   │
├──────────────────┴───────────────────┤
│                   │                   │
│      p4           │       p5          │
│                   │                   │
└──────────────────────────────────────┘
```

## 交互式调用

当你说「生成排版图」或「用这个 skill」时，我会问你：

1. **图片文件夹路径** — PPT 导出图片所在的文件夹
2. **布局样式** — 目前可用：
   - `layout1`：左侧9缩略图 + 右侧3大图（适合12页PPT）
   - `layout2`：三层瀑布流（适合5页PPT）
3. **背景色** — 可选，默认 #f3dfc6

### 示例对话

```
你：帮我生成排版图，图片在 G:/PPT/颁奖典礼/
我：好的，请选择布局样式：
    1. layout1：左侧9缩略图 + 右侧3大图
    2. layout2：三层瀑布流

你：选 layout2
我：好的，正在生成排版图...
```

## 命令行调用

```bash
# 列出所有布局
python ~/.claude/skills/dp-xhsppt-output_layout/ppt_layout.py --list

# 使用 layout1（默认）
python ~/.claude/skills/dp-xhsppt-output_layout/ppt_layout.py "G:/PPT导出图片/"

# 指定布局
python ~/.claude/skills/dp-xhsppt-output_layout/ppt_layout.py "G:/PPT导出图片/" -l layout2
```

## 输入要求

- 图片格式：PNG、JPG、JPEG
- 图片尺寸：建议 16:9
- 图片命名：按顺序排列

## 输出

- 格式：PNG
- 尺寸：1242 × 1660px
- 位置：与输入图片同目录
